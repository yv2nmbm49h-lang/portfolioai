Training Data: All of the text that is used to teach the language model
Parameters: The weights of the neural network
Tokens: The different parts of text that a model uses
Next-token prediction: The task that the language model is given
Context window: The amount of text the language model can see when it’s generating a response
Hallucination: When the language model gives you information that you think is real but isn’t
Fine-tuning: A finer training stage after the normal language model is generated
RLHF: Reinforcement Learning from Human Feedback

How A Large Language Model Works:
A large language model works by taking in data from a user and using it to generate a result. The way that it does this is through parameters and tokens along. RLHF is also necessary so that there isn’t any hallucination from the model.
